# Recipe — Parallel-Set Modules

The shape for a series of sibling modules that share the same structural role. The mid-arc teaching shape: each module covers one sibling concept, reusing the SAME visual shape to communicate "these are parallel."

## Proven in
- Modules 04, 05, 06, 07 — Foundation Model, Perception, Memory, Planning (four siblings introducing the agent stack's context components)

## Combines
- `editorial-spread` (eyebrow + headline-with-italic-period — content changes per module, shape doesn't)
- `chat-panel` (right-side IDE-style chat — content changes per module)
- `indicator-strip` (bottom — one card brighter than others, indicating the current subject; cross-fade as the next module takes over the highlight)

## Beat structure (per module)

```
0–3s     Open with strip in its handed-off state from previous module (current subject card brightening)
3–8s     Editorial spread (eyebrow + headline) drifts in on the left
8–25s    Chat panel demonstrates the concept (prompt + streamed response)
25–55s   Teaching beat — could be additional inline visuals, callouts, satellite elements
55–65s   Handoff — current subject card dims; next subject card brightens; Tools card subtly intensifies (foreshadowing Module 08)
```

## Energy curve
- Steady educational pace throughout.
- Small lifts at the chat panel reveal and the handoff.
- No peaks — the peak is reserved for Module 08 (the arrival).

## Why this shape
When four modules play the same structural role, viewers learn the shape and read each module faster. The repeated shape becomes a comfort signal: "another sibling, here we go."

## Critical discipline
**Do NOT reuse the parallel-set shape for non-sibling beats.** When you transition out of the set (e.g., to a climax), the shape MUST change to signal the role change. See `arrival-module.md`.

## Foreshadow technique (load-bearing)
Across all four parallel-set modules, the **Tools card in the indicator strip stays bright** while the other three dim. This is the load-bearing decision that makes Module 08's arrival feel earned — viewers have seen Tools "on stage" for 4 modules without it being explained. When Module 08 finally promotes it to centerpiece, the payoff lands.

Plant the seed early. Let it stay visible. Cash it at the climax.
