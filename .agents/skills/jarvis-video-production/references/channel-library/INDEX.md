# Channel Library Index

This folder mirrors the reusable library from the older Claude-local `byrddynasty-video-production` skill. Use it when a video needs a specific reusable shape, interaction pattern, channel identity rule, recipe, or anti-pattern.

## Folders

- `reference/` — shape catalog, asset standards, tone matrix, and anti-patterns.
- `vocabulary/` — reusable interaction and composition patterns such as terminal sessions, VS Code simulation, annotated screenshots, diff reveals, bento pans, cursor reveals, and ambient cutaways.
- `recipes/` — higher-level combinations of shapes and vocabulary.
- `channel-identities/` — channel-specific identity tokens.

## Use

Read `vocabulary/README.md` first when selecting a visual technique. Read only the specific pattern file needed for the current scene.

## Maintenance

When a new production pattern is proven in a future episode, add it here or update the matching pattern file. Do not leave proven patterns only in a one-off episode folder.
