## Analysis Methodology

**Framework:** Chris Vermeulen's Asset Revesting 4-Stage Model
**Scoring:** Composite (Trend 40%, Momentum 30%, Position 30%)
**Data:** Alpaca API via IEX feed (256 trading days)

**Stage Definitions:**
- Stage 1: Accumulation (basing, prepare to buy)
- Stage 2: Markup (uptrend, buy and hold)
- Stage 3: Distribution (topping, prepare to sell)
- Stage 4: Decline (downtrend, avoid or short)

- 2x-3x daily reset can cause tracking error
- Use only in strong trends
- Reduce position size vs unleveraged

**Inverse ETFs:**
- Benefit from market declines
- Only use in bearish markets or for hedging
- Not for long-term holding

## Scoring Methodology (Detailed)

### Trend Strength (40 points maximum)

**SMA Alignment (40 points total):**
- Price > 200 SMA: 10 points
- Price > 150 SMA: 10 points
- Price > 50 SMA: 10 points
- 50 SMA > 150 SMA > 200 SMA: 10 points

**Scoring Logic:**
```python
trend_score = 0
if price > sma_200:
    trend_score += 10
if price > sma_150:
    trend_score += 10
if price > sma_50:
    trend_score += 10
if sma_50 > sma_150 > sma_200:
    trend_score += 10
```

### Momentum (30 points maximum)

**RSI (15 points):**
- RSI 50-70: 15 points (ideal)
- RSI 40-50: 10 points (acceptable)
- RSI 70-80: 8 points (strong but overbought)
- RSI 30-40: 5 points (weak)
- RSI >80 or <30: 0 points (extremes)

**MACD (15 points):**
- MACD > Signal by >0.5: 15 points (strong bullish)
- MACD > Signal by 0-0.5: 12 points (bullish)
- MACD < Signal by 0-0.5: 8 points (neutral)
- MACD < Signal by >0.5: 0 points (bearish)

### Position vs SMAs (30 points maximum)

**Distance Above SMAs (scaled):**

Each SMA (10 points max):
- 0-2% above: 10 points (ideal)
- 2-5% above: 8 points (good)
- 5-10% above: 5 points (extended)
- >10% above: 3 points (very extended)
- Below SMA: 0 points

**Logic:**
```python
def score_position(price, sma):
    pct_above = ((price - sma) / sma) * 100
    if pct_above < 0:
        return 0
    elif pct_above <= 2:
        return 10
    elif pct_above <= 5:
        return 8
    elif pct_above <= 10:
        return 5
    else:
        return 3

position_score = (
    score_position(price, sma_50) +
    score_position(price, sma_150) +
    score_position(price, sma_200)
)
```

## Stage Classification Logic

### Stage 2 (Markup) Criteria

**Required (all must be true):**
- Price > 50 SMA
- 50 SMA > 150 SMA
- 150 SMA > 200 SMA

**Confidence Modifiers:**
- High (85-100%): RSI 50-70, MACD > Signal, all SMAs rising
- Medium (70-84%): RSI 40-70, MACD neutral/positive
- Low (50-69%): Mixed signals, some criteria met

### Stage 1 (Accumulation) Criteria

**Characteristics:**
- Price within ±5% of 150 SMA
- SMAs flattening (not clearly trending)
- RSI 40-60
- Low volatility

### Stage 3 (Distribution) Criteria

**Warning Signs:**
- Price above 150 SMA but...
- MACD bearish divergence (MACD < Signal)
- RSI > 70 (overbought)
- SMAs starting to roll over

### Stage 4 (Decline) Criteria

**Required:**
- Price < 50 SMA
- 50 SMA < 150 SMA (preferably < 200 SMA too)
- Downtrend confirmed

## Data Quality Standards

### Required Data Points

For each ETF, must have:
- Current price
- At least 200 trading days of history (for 200 SMA)
- All SMAs calculated (20, 50, 150, 200)
- RSI (14-day)
- MACD and Signal line

### Handling Missing Data

**If 200 SMA unavailable:**
- Note in report: "⚠️ Limited data (XX days)"
- Use 150 SMA as primary trend indicator
- Reduce trend score proportionally
- Flag as "incomplete analysis"

**If any indicator fails:**
- Note in individual ETF analysis
- Use available indicators only
- Reduce confidence level
- Do NOT skip the ETF entirely

### Data Freshness

- Data should be <24 hours old
- Note data source and timestamp in report
- Alpaca IEX feed: 15-minute delay (acceptable)

## Edge Cases

### Leveraged ETFs (SSO, UPRO, SQQQ, SPXS, TBT)

**Special Considerations:**
- Calculate stage normally BUT
- Add warning about daily reset/decay
- Recommend smaller position sizes
- Only recommend in strong trends (Stage 2 confidence >85%)

**Example Note:**
```markdown
⚠️ **Leveraged ETF Warning:** This is a 3x leveraged product. Daily reset can cause tracking error over time. Use only for short-term tactical trades in strong trends. Reduce position size vs unleveraged equivalents.
```

### Inverse ETFs (SQQQ, SPXS, TBT)

**Logic Inversion:**
- Stage 2 in SQQQ = QQQ in Stage 4 (weak)
- Only recommend if underlying is truly weak
- Flag as "bearish positioning" not "bullish"

**Example:**
```markdown
### SQQQ (-3x QQQ) - Inverse Nasdaq

**Underlying QQQ Status:** Stage 4 (Decline)
**SQQQ Stage:** 2 (Benefits from QQQ weakness)
**Use Case:** Bearish bet on tech or portfolio hedge
**Risk:** 3x leverage, high volatility
**Recommendation:** CONSIDER (only if bearish on tech)
```

### Cash Equivalent (BIL)

**Always:**
- List as Stage 2 (safe haven)
- Score: 100/100 (Grade A)
- Recommendation: HOLD (defensive allocation)
- Note: "Cash equivalent - use for defensive positioning"
