# Quick Execution Checklist

Use this checklist when creating video images. Complete all steps in order.

---

## Pre-Flight Check

- [ ] Script is available and complete
- [ ] SKILL.md has been read
- [ ] Reference examples location noted
- [ ] Ready to create image prompts

---

## Step 1: Analyze Script

- [ ] Read script beginning to end
- [ ] Identify all major concepts (key ideas to visualize)
- [ ] Note data points (statistics, percentages, numbers)
- [ ] Understand narrative flow (story arc)
- [ ] Count natural segments (how many distinct concepts?)

**Output:** List of concepts to visualize

---

## Step 2: Create Sequence Map

- [ ] Assign simple sequential numbers (001, 002, 003...)
- [ ] One concept = one number (no 001A, 001B!)
- [ ] Map each number to script segment
- [ ] Verify numbers match script order
- [ ] Confirm easy 1:1 matching for video production

**Output:** Numbered list mapping concepts to script segments

---

## Step 3: Plan Each Image

For each image number, determine:

- [ ] **Visual type** (progression, comparison, data, cycle, illustration)
- [ ] **Main elements** (what visuals tell this story?)
- [ ] **Text content** (headline, labels, data)
- [ ] **Color approach** (what colors and why?)
- [ ] **Layout structure** (horizontal flow, split, circular, zoned)

**Output:** Visual plan for each image

---

## Step 4: Craft Prompts

For each image, write complete prompt including:

- [ ] **Concept** (one sentence - what story?)
- [ ] **Visual elements** (3-5 main components)
- [ ] **Text on image** (headline LARGE, labels brief)
- [ ] **Color palette** (colors with meaning)
- [ ] **Style notes** (mood, tone, specific metaphors)
- [ ] **Generation prompt** (complete DALL-E/Midjourney prompt)

**Use IMAGE-PROMPT-TEMPLATE.md for structure**

---

## Step 5: Quality Check Each Prompt

### Visual Clarity ✓

- [ ] Concept is clear without narration
- [ ] Visual hierarchy guides eye
- [ ] Image tells a story
- [ ] One main concept (not overcrowded)

### Text Balance ✓

- [ ] Headline is LARGE (largest text element)
- [ ] Body text minimal (under 15 words per section)
- [ ] Text supports visuals (doesn't replace them)
- [ ] Font sizes create clear hierarchy

### Color & Design ✓

- [ ] Colors are vibrant (not drab)
- [ ] Colors have purpose/meaning
- [ ] NOT just boxes with text
- [ ] Artistic and engaging style

### Simplicity ✓

- [ ] One concept per image
- [ ] Clean layout
- [ ] Easy to understand at a glance
- [ ] Not over-detailed

### Production Ready ✓

- [ ] Simple sequential number (001, 002, 003...)
- [ ] Maps to script segment clearly
- [ ] Appropriate aspect ratio (16:9 for video)
- [ ] Includes all necessary elements

---

## Step 6: Reference Check

Compare against best examples:

- [ ] **001A (Video 3)** - Circular flow quality?
- [ ] **001CB (Video 3)** - Clean stage progression?
- [ ] **002AB (Video 3)** - Prominent numbers/timeline?
- [ ] **003B (Video 2)** - Visual storytelling depth?
- [ ] **006A (Video 2)** - Data chart clarity?

**Ask:** Would this fit alongside the reference examples?

---

## Step 7: Final Deliverable

Package includes:

- [ ] **Sequence map** (all numbers with concept names)
- [ ] **Complete prompts** (one per image, fully detailed)
- [ ] **Production notes** (style consistency, timing, aspect ratio)
- [ ] **Source credits** (if applicable)

**Format:** Clean markdown document, ready to copy-paste

---

## Style Quick Reference

### DO ✅

- Rich vibrant colors with meaning
- LARGE prominent headlines
- Minimal body text (2-3 lines max)
- Visual metaphors and storytelling
- Illustrations, icons, diagrams
- Arrows showing progression/flow
- Clear visual separation (zones, sections)
- Simple sequential numbering

### DON'T ❌

- Drab boring colors
- Text-heavy slides
- Paragraphs of explanation
- Generic stock photo style
- Overcrowded layouts
- Sub-numbering (001A, 001B, 001C)
- Multiple concepts in one image
- Vague unclear visuals

---

## Visual Type Decision Tree

**Showing progression over time?**
→ Timeline/Stage progression (horizontal flow, arrows)

**Comparing two things?**
→ Split screen (side-by-side or top/bottom)

**Presenting data/statistics?**
→ Data visualization (hero number or chart with zones)

**Showing a repeating pattern?**
→ Circular cycle (central concept, surrounding stages)

**Explaining a concept?**
→ Concept illustration (central idea, supporting elements)

---

## Color Decision Guide

**Positive/Optimal/Growth?** → Green
**Caution/Attention/Transition?** → Yellow/Gold
**Warning/Energy?** → Orange
**Danger/Decline/Critical?** → Red
**Trust/Data/Professional?** → Blue
**Premium/Advanced?** → Purple
**Neutral/Baseline?** → Gray

---

## Text Hierarchy Quick Guide

1. **Main Headline** - LARGEST (60-80pt equivalent)
   - 3-8 words
   - ALL CAPS often
   - Bold weight

2. **Key Data/Numbers** - VERY LARGE (50-70pt)
   - Prominent statistics
   - Bold weight
   - Can use color accent

3. **Subheadings** - Medium-Large (30-40pt)
   - 4-10 words
   - Section labels

4. **Labels/Annotations** - Medium (20-30pt)
   - 1-3 words each
   - Brief context

5. **Source Citations** - Small (12-16pt)
   - Bottom of image
   - Credibility element

---

## Common Mistakes to Avoid

❌ **Sub-numbering** (001A, 001B) - Use 001, 002, 003 instead
❌ **Text walls** - Keep text under 15 words per section
❌ **Plain boxes** - Use illustrations, gradients, visual elements
❌ **Unclear visuals** - Story should be clear without narration
❌ **Too many concepts** - One concept per image
❌ **Drab colors** - Use vibrant, purposeful colors
❌ **Missing hierarchy** - Make headlines LARGE and prominent

---

## Emergency Style Check

If unsure about a prompt, ask:

1. **Can I understand this without narration?** (Visual clarity)
2. **Is the headline the largest text?** (Text hierarchy)
3. **Are there vibrant colors with meaning?** (Color purpose)
4. **Is it simple and clean?** (Not overcrowded)
5. **Does it look like the reference examples?** (Style match)

If any answer is "no" → revise the prompt

## Time Estimate

**Typical video (8-12 concepts):**
- Script analysis: 10-15 minutes
- Sequence mapping: 5-10 minutes
- Prompt crafting: 15-20 minutes per image
- Quality check: 5-10 minutes total
- **Total: 2-4 hours** for complete set

**Efficiency tip:** Do all analysis first, then craft all prompts, then quality check all at once.
