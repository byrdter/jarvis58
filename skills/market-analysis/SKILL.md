# Market Analysis Skill

Perform 4-stage market analysis on a given symbol using the Asset Revesting methodology.

## Mandatory Reading

Before executing this skill, read:
1. `design-requirements.md` in this directory
2. `context/projects/investments/CLAUDE.md` for methodology details

## Purpose

Analyze a stock, ETF, or index to determine:
1. Current market stage (1-4)
2. Stage transition signals
3. Technical indicator status
4. Actionable recommendation

## Input

- **Symbol** (required): The ticker symbol to analyze (e.g., QQQ, SPY, AAPL)
- **Timeframe** (optional): Daily (default), Weekly, or Monthly
- **Data Source** (optional): If real data available, specify source

## Process

### Step 1: Gather Data

**Real Data (Recommended):**

Use the market-data CLI tool to fetch live data:

```bash
# Fetch all technical indicators at once
DATA=$(/Users/terrybyrd/Library/CloudStorage/Dropbox/jarvis/cli-tools/jarvis-price indicators SYMBOL --json)

# Parse JSON data using jq or read directly in analysis
# Example:
CURRENT_PRICE=$(echo $DATA | jq -r '.current_price')
SMA_50=$(echo $DATA | jq -r '.sma_50')
SMA_150=$(echo $DATA | jq -r '.sma_150')
SMA_200=$(echo $DATA | jq -r '.sma_200')
RSI=$(echo $DATA | jq -r '.rsi_14')
MACD=$(echo $DATA | jq -r '.macd')
MACD_SIGNAL=$(echo $DATA | jq -r '.macd_signal')
```

**Data Source:** Yahoo Finance via yfinance (15-20 min delay)

**If CLI tool fails or no internet:**
- Use sample data or data provided by user
- Clearly state that analysis is based on sample/provided data
- Mark report with ⚠️ warning at top

### Step 2: Calculate Indicators

**With Real Data:**
The CLI tool provides pre-calculated indicators. Review and interpret:
- **Current values** - From JSON response
- **Trend** - Compare current SMA positions (if price > SMA and SMA rising = bullish)
- **Signal** - Apply Asset Revesting criteria from design-requirements.md

**Without Real Data:**
Calculate indicators manually or use provided data

### Step 3: Determine Stage
Apply the 4-stage framework:
- Check moving average alignment
- Evaluate price position relative to MAs
- Assess momentum indicator signals
- Consider volume patterns
- Assign stage (1, 2, 3, or 4)

### Step 4: Identify Transitions
Look for signals that suggest stage may be changing:
- Divergences between price and indicators
- Moving average crossovers pending
- Volume anomalies
- Support/resistance tests

### Step 5: Generate Recommendation
Based on stage and transitions:
- Stage 1: "Monitor for breakout"
- Stage 2: "Accumulate on pullbacks" or "Hold with trailing stop"
- Stage 3: "Tighten stops, reduce exposure"
- Stage 4: "Avoid or sell existing positions"

### Step 6: Calculate Composite Score
Using the scoring system from methodology:
- Technical Score (40%)
- Relative Strength Score (30%) 
- Stage Score (30%)
- Total: 0-100
