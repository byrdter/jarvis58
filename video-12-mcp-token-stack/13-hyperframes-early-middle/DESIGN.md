This batch covers the early-middle argument after the first proof scenes. It should feel like six directed visual events, not a presentation deck.

## Scene Types

- Segment 004: warning machine showing what Tool Search does and does not fix.
- Segment 005: animated benchmark rig with bars, gauges, and stress-test failure zone.

## Constraints

- No side avatars.
- No generic static panels as the main event.
- Every scene must change state at least three times.
- Use large readable text and avoid text-on-text overlap.
