# Video 12 Animated Movie Proof Design

## Style Prompt

Build this proof as a kinetic engineering documentary made from full-screen animated visual scenes. Each subsegment should feel like a directed shot: large objects, clear spatial metaphor, minimal readable labels, strong movement, and a decisive final hero frame. The generated stills in `09-stills/TestFolder` are concept art and storyboard references, not pixel targets.

## Colors

- Canvas: `#07101d`
- Deep field: `#0b1424`
- Glass panel: `rgba(15, 30, 52, 0.84)`
- Context cyan: `#34e8ff`
- Search blue: `#5aa7ff`
- Success green: `#68f59b`
- Warning amber: `#ffbf4d`
- Waste red: `#ff4b64`
- Code purple: `#9b6cff`

## Typography

- Display/UI: `Inter, Arial, sans-serif`
- Mono/code: `JetBrains Mono, Consolas, monospace`
- Metrics: `Orbitron, Arial Black, sans-serif`

## Motion Rules

- Every subsegment must have an entrance, internal motion, and exit.
- Prefer object movement over text changes: streams, slams, sweeps, wipes, boundary crossings, meter fills, matrix scans.
- Labels must be large and sparse. Avoid recreating dense AI still text literally.
- Use generated stills as a design reference only; build the scene as HyperFrames layers.
