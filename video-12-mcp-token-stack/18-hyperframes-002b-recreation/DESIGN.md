# 002B HyperFrames Recreation Design

Reference: `09-stills/TestFolder/002B.png`

Goal: create a 10-second object-built HyperFrames proof that resembles the "Context Memory Before the Task" dashboard.

Core motion:

- Orange safety frame and warning icon ignite first.
- Left readiness panels slide in from off-screen left.
- Right problem-space panels slide in from off-screen right.
- Main heading and "Current Total: 55,000 Tokens" land from above.
- Token axis and grid fade in behind the stack.
- Six tool-definition bars stack into the center one after another, with horizontal speed-line streaks inside each bar.
- Completed dashboard holds for the final seconds so it can support the voiceover.

Reusable motion ideas:

- `panel-snap-in`: side diagnostic panels arrive from outside frame and settle with a small bounce.
- `definition-stack`: horizontal bars enter from alternating sides and lock into a vertical stack.
- `metric-glow-count`: big numeric metric scales and glows as it becomes the dominant data point.
- `readiness-frame`: perimeter border pulses as a pre-task system state indicator.
