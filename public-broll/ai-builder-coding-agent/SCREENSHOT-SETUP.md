# Public Screenshot Setup

Use this folder for a safe YouTube screenshot about how AI builders work with coding agents.

## Open These Files

- `AGENT-TASK.md`
- `demo_quality_helper.py`
- `terminal-output.txt`

## Recommended Layout

- Left editor group: `AGENT-TASK.md`
- Right editor group: `demo_quality_helper.py`
- Bottom panel or second small editor: `terminal-output.txt`

## Hide Before Capture

- File explorer/sidebar
- Breadcrumbs
- Status bar
- Git/source-control panel
- Terminal prompt if it shows a username, machine name, or real folder
- Any tabs for `.env`, private notes, client files, or real project work

## Framing

Capture a 1920x1080 frame with only the three safe files visible. Keep the sidebars hidden so the shot reads as a coding-agent workflow without exposing the surrounding workspace.
