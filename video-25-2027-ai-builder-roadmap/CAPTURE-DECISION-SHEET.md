## Do Not Capture

- `.env` files
- API keys, billing pages, account pages, tokens, cookies, or secrets
- Private client/project names
- Raw local folder trees that reveal sensitive personal structure
- Unreviewed notes, transcripts, or source lists that were not intended for YouTube

## Recommendation

Proceed with the sanitized mock now. After the mock is reviewed, replace only 2-3 surfaces with real local captures if they clearly improve credibility. The episode should feel personal because JARVIS is the operating example, not because every screen is literal.
