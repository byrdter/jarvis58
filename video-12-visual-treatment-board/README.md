# Video 12 Visual Treatment Board

This is the clean restart folder for Video 12 planning.

The goal is not to force the video through one automation method. The goal is to choose the strongest visual treatment for each teaching beat before writing or finalizing VO, prompts, HyperFrames scenes, Remotion assemblies, or HeyGen segments.

## Current Planning Artifacts

- `01-planning/VISUAL-TREATMENT-BOARD-FIRST-10.md` - detailed treatment options and recommendations for subsegments 002A through 010E.

## Working Principle

For each subsegment, decide the visual direction first:

1. What is the idea?
2. What would the viewer see?
3. What moves or changes?
4. Which tool is best for producing that shot?
5. What is the recommended direction?

Only after that should we write final VO, create Veo prompts, build HyperFrames/Remotion elements, or record source material.
