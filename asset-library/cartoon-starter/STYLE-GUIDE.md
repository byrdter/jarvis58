# Cartoon Starter Style Guide

## Visual Language

- Flat 2D cartoon explainer style.
- Thick dark rounded outlines.
- Soft paper texture rather than glossy gradients.
- Limited, warm palette with blue, green, coral, gold, cream, purple, and dark ink.
- Simple rounded anatomy and expressive faces.
- No photorealism, no tiny detail, no complex shading.

## Asset Prompt Recipe

Use this when generating one-off stills or new reusable assets:

```text
Flat 2D cartoon explainer asset, thick rounded dark outline, simple rounded shapes, limited warm palette, subtle paper texture, clean transparent-background composition, no text, no watermark, readable at YouTube 720p, designed as a reusable layered explainer-video asset.
```

For character variants:

```text
Same flat 2D cartoon character as the reference, full body, transparent background, thick rounded dark outline, simple shapes, consistent palette, pose: [POSE], expression: [EXPRESSION], no text, no background, no watermark.
```

## Motion Rules

- Use pose swaps instead of complex full-body acting.
- Animate props separately from characters.
- Keep motion readable: one primary action per beat.
- Add subtle drift or texture movement when holding a scene.
- Use diagram builds for explanation, not decorative motion.
